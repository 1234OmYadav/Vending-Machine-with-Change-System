# Vending-Machine-with-Change-System
This project is based on designing a vending machine with a change system using Verilog HDL and the concept of a Finite State Machine (FSM).

The machine accepts coins of ₹5 and ₹10, and the cost of one bottle is ₹15. I designed different states to represent the amount inserted by the user. For example, the initial state represents 0 Rs, and as the user inserts coins, the system transitions to different states like 5 Rs or 10 Rs.
